# AI Agent Setup Guide

This guide will help you set up the AI Agent on your Raspberry Pi 5.

## Prerequisites

- Raspberry Pi 5 (4GB or 8GB recommended)
- MicroSD card 32GB+ (Class A1/A2 recommended)
- Raspberry Pi OS 64-bit (Bookworm) installed
- Stable internet connection
- API keys for AI models (Kimi, OpenAI, Claude, etc.)

## Quick Start

### 1. Clone the Repository

```bash
# SSH into your Raspberry Pi
ssh pi@raspberrypi.local

# Clone the repository
git clone https://github.com/YOUR_USERNAME/ai-agent-raspberry-pi5.git
cd ai-agent-raspberry-pi5
